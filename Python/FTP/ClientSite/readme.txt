Prerequisite environment:
Python (version 3 or above)

Step to start the Client Site of FTP:
1. copy the path of "bin folder"
2. open cmd.exe or Windows PowerShell
3. change directory in cmd.exe or Windows PowerShell to the path just copied
4. follow the syntax listed below:
   If you registered before:
    python ClientSiteLauncher.py start -i (the ip address the Server uses) -p (the port the Server uses)
   Otherwise
    python ClientSiteLauncher.py register -i (the ip address the Server uses) -p (the port the Server uses)
